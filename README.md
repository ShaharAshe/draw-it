## Installation and Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/ShaharAshe/draw-it.git
   ```

2. Navigate to the project directory:
   ```bash
   cd draw-it
   ```

4. Start drawing, saving your artwork, or erasing as needed!
